# Roll Initiative!
----
## A tabletop roleplaying game assistant

#### Key Features:
* Simple Register/Login
* Character creator with Race, Class, & Background look-up (Spells & Equipment soon to follow)
* D4 - D100 Dice Roller
* API data access

#### In Progress:
* Quest logs & Session notekeeper
* Exportable character sheets
* Campaign notator
* Game Master tools
